# Memory Matching Game

A browser-based card-flipping memory game built as a university assignment.
Built with vanilla HTML, CSS, and jQuery — no frameworks or build tools.

## Features

- 4×4 grid of shuffled cards, randomized every game
- Flip logic with a 2-card limit and mismatch delay
- Move counter and live timer
- Match and wrong sound effects
- Win screen with final stats
- Fully responsive layout

## Tech Stack

- HTML5
- CSS3 (transitions & animations)
- jQuery 3.7.1
